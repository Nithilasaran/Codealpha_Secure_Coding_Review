# CodeAlpha Cyber Security Internship

## Task: Secure Coding Review

### Project Overview

This project demonstrates a secure coding review of a Python login system.

The application was analyzed for common security vulnerabilities and recommendations were provided to improve security.

### Vulnerabilities Identified

1. Hardcoded Credentials
2. Weak Password Usage
3. No Input Validation
4. No Account Lockout Mechanism
5. Plain Text Password Storage

### Security Improvements

- Password Hashing using bcrypt
- Strong Password Policies
- Input Validation
- Secure Authentication Practices

### Technologies Used

- Python
- bcrypt
